What is taskescape?

A web based task manager to help users to keep track of their tasks. You will be able to manage your tasks in 2 different categories, personal and work. You will be abled to view all tasks in 2 different views, list and board view. With this system you will be able to keep track of all your tasks with ease. Moreover, You will be notified about tasks that are due in less then a week.

Guide to using taskescape

Create a task

  • Navigate to the task list page using the top nav bar.
  • Click on the category button you want to create a new task in.
  • Select 'Upcoming' from the drop list.
  • Click the 'Create Task' button, located in the top right
  • Fill out the form
  • To finish creating click submit task

Edit a task

  • To edit a task click the 'Edit' button, located on each task in the Upcoming section
  • A prefiled form wil be displayed, correct the details
  • To finish editing, click the submit button

Mark as done

  • Marking a task as completed can be done by clicking the 'Done' button located on each task in the Upcoming section
  • To view the compeleted tasks, using the category button select 'Done' from the drop lists
  • All completed task will be shown in the Done section

Delete a task

  • To delete a task navigate to the done section
  • Click the 'Delete' button
